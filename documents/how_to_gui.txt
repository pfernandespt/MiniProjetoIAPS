Para criar uma aplicação com GUI e scripts executando indefinidamente em MATLAB, você pode seguir as seguintes etapas:

Crie uma interface gráfica do usuário (GUI) usando o MATLAB's GUIDE (Graphical User Interface Development Environment). A GUI permitirá que os usuários interajam com sua aplicação e executem as funções necessárias.

Escreva os scripts que precisam ser executados indefinidamente em segundo plano. Para fazer isso, você pode criar uma função que contém o código que precisa ser executado e usar a função "while" para executar o código indefinidamente.

Para compartilhar variáveis entre os scripts e a GUI, você pode usar variáveis globais. As variáveis globais podem ser acessadas por todos os scripts e funções em sua aplicação.

Para permitir que os scripts interajam com a GUI, você pode usar a função "setappdata" para armazenar os valores das variáveis compartilhadas na GUI e a função "getappdata" para recuperar os valores armazenados na GUI.

Finalmente, você pode adicionar a lógica necessária para cada botão da GUI para executar a função correta que altera as variáveis compartilhadas. Você também pode adicionar a lógica para atualizar a GUI com os valores atualizados das variáveis compartilhadas.

Para um exemplo de como fazer isso, você pode consultar a documentação do MATLAB sobre a criação de aplicativos com GUI: https://www.mathworks.com/help/matlab/creating_guis/what-is-a-gui.html
